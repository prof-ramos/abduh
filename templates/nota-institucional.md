# Template de Nota Institucional — ASOF

---

## Estrutura

### Cabeçalho

```
NOTA INSTITUCIONAL
[Número da nota, se houver]
[Local], [dia] de [mês] de [ano].
```

### Título

```
[Título conciso da nota]
```

### Corpo

```
A Associação Nacional dos Oficiais de Chancelaria do Serviço Exterior
Brasileiro (ASOF) vem a público informar que...

1. [Fato — o que ocorreu]
2. [Posição institucional — qual a posição da ASOF]
3. [Encaminhamento — próximos passos, se aplicável]
```

### Fecho

```
[Contato para imprensa, se aplicável]
```

---

## Campos Preenchíveis

| Campo | Exemplo | Obrigatório |
|---|---|---|
| `[Número]` | 001/2026 | opcional |
| `[Local]` | Brasília | sim |
| `[dia]` | 17 | sim |
| `[mês]` | junho | sim |
| `[Título]` | Posicionamento da ASOF sobre... | sim |
| `[Fato]` | Parágrafo descritivo | sim |
| `[Posição]` | Parágrafo de posicionamento | sim |
| `[Encaminhamento]` | Parágrafo de próximos passos | condicional |

---

## Regras de Estilo

1. **Tom institucional** — formal, sem linguagem publicitária
2. **Sem emojis** no texto da nota
3. **Validação humana obrigatória** para notas públicas
4. **Risco reputacional** — acionar Strategy Chief antes da publicação
5. **LGPD** — não expor dados pessoais de associados
6. **Citar sempre "ASOF" por extenso** na primeira menção
